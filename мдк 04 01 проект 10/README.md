# Система управления личными финансами

Архитектурное описание приложения для учета доходов и расходов с UML и ER-диаграммами.

## Функциональные возможности

- Управление доступом: регистрация, авторизация, управление профилем.
- Учет операций: счета в разных валютах, транзакции с привязкой к категориям.
- Бюджетирование: лимиты расходов на периоды, контроль выполнения.
- Аналитика: графики, статистика, экспорт данных.
- Администрирование: управление пользователями.

## Технологический стек

- Backend: Controller-Service-Database (ООП)
- Database: реляционная СУБД (SQL)
- Frontend: UI-интерфейс с визуализацией отчетов
- Архитектура: модульная (аутентификация, транзакции, отчетность)
