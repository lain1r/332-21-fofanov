Система управления личными финансами
   Проект представляет собой архитектурное описание приложения для учета доходов и расходов, документированное с помощью UML и ER диаграмм.

Функциональные возможности
   Управление доступом: Регистрация, авторизация и управление профилем пользователя.
   Учет операций: Создание счетов в разных валютах, добавление и редактирование транзакций с привязкой к категориям.
   Бюджетирование: Установка лимитов расходов на определенные периоды и контроль их выполнения.
   Аналитика: Формирование графиков, просмотр статистики и экспорт данных.Администрирование: Управление пользователями в рамках системы.
   
Технологический стек
   Backend: Объектно-ориентированная среда разработки с разделением на контроллеры и сервисы (Controller-Service-Database).
   Database: Реляционная СУБД (SQL) для хранения данных о пользователях, счетах и операциях.
   Frontend: UI-интерфейс для взаимодействия с пользователем и визуализации отчетов.Архитектура: Модульная структура, включающая компоненты аутентификации, обработки транзакций и отчетности.
